# nixos-cli

> Unified NixOS management tool — drop-in replacement for nixos-rebuild with TUI for generations and options.

Source: `github:nix-community/nixos-cli`

### Purpose

All-in-one NixOS administration tool. Consolidates functionality from multiple utilities into a single cohesive CLI with terminal UIs for generation management and system option exploration.

### Key Features

- Drop-in replacement for `nixos-rebuild`
- TUI for browsing and managing system generations
- Interactive system option explorer
- Written in Go with comprehensive man pages

### Use Cases

- **System rebuilds**: more intuitive interface than raw nixos-rebuild
- **Generation management**: interactive browsing and cleanup
- **Option discovery**: explore available NixOS options visually
